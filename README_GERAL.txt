NEXTERP - PRIMEIRA VERSAO MVP

Este pacote contém:
1. Backend em Python com FastAPI
2. Banco SQLite inicial
3. Frontend em React
4. Cadastro de produtos
5. Dashboard básico
6. Calculadora de preço com:
   - custo do produto
   - taxa marketplace
   - frete
   - antecipação
   - lucro desejado

PRÓXIMOS MÓDULOS:
- Login e usuários
- Integração com Shopee
- Integração com Mercado Livre
- Relatório mensal de lucro
- Importação automática de vendas
- Controle de estoque por marketplace
- Cadastro de custos fixos por mês
